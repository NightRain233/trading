基于我们之前的讨论，可以将你的交易系统总结为以“多周期共振”为基石，以“金叉为池、回踩为买点、趋势跟踪为卖点”的完整策略。

以下是策略的核心要点梳理：

一、 核心逻辑与选股框架
核心理念：顺势而为，只做多周期共振的标的。

战略前提（筛池子）：

周线级别保护：只考虑 周线MACD处于金叉状态 且 股价位于周MA5上方 的标的。这确保了操作在大的上升趋势中。
日线级别趋势：重点关注 日线EMA20与EMA50刚刚形成金叉（或金叉后10-15天内） 的标的。这代表了中期趋势的启动，构成了你的观察池。
二、 买入信号
在观察池的基础上，寻找日线级别的回调买入机会，即“回踩确认”。

买入触发条件：股价在上升途中回踩短期均线（EMA5 或 EMA10）。

确认信号（增强信号可靠性）：

价格确认：最低价曾触及或轻微跌破均线，但收盘价成功收在均线之上。
成交量确认：回踩当天成交量明显萎缩（例如，低于20日均量的60%-80%），表明抛压衰竭。
K线形态确认（加分项）：回踩日K线出现长下影线、十字星等企稳形态。
中期趋势保护：回踩的最低点没有跌破EMA20，确保中期趋势完好。
均线方向：回踩的短期均线（EMA5/10）本身保持向上倾斜。
三、 卖出信号
卖出策略与买入逻辑对称，旨在保护利润并控制风险。建议采用分级止盈止损的方式。

初始止损（硬止损）：

买入后，若股价跌破关键支撑位（如买入价下方-5%~-8%，或明确跌破EMA20），无条件离场。这是风险控制的第一道防线。

移动止盈（跟踪止损）：

当股价脱离成本区（例如盈利超过10%）后，将防守线从固定止损上移至趋势跟踪均线，如EMA10 或 EMA20。只要股价收盘不破该均线，就一直持有，让利润奔跑。

趋势转弱信号（辅助离场）：

指标死叉：日线MACD出现死叉。
顶背离：股价创新高，但MACD或RSI指标未创新高。
成交量异常：出现放量滞涨或缩量新高。
最终清仓（战略离场）：

股价跌破EMA50，或周线级别趋势走坏（如跌破周MA5，或周线MACD死叉），彻底清仓。

四、 策略的量化实现简述
Python筛选逻辑：

遍历股票池，筛选出满足“周线条件 + 日线EMA20/50金叉（10-15天内）”的标的。
对筛选出的标的，调用回踩检测函数，检查最近几个交易日内是否满足“价格回踩短期均线 + 缩量 + 收盘收回 + 不破中期均线”的条件。
输出符合买入信号的股票列表，进行人工复核。
卖出监控：

每日持仓检查，若触发上述任何一条卖出规则（特别是移动止损线），则发出卖出信号。
