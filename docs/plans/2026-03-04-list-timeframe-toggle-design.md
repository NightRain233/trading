# 列表页日/周切换设计

## 目标
在列表页新增日/周切换按钮，使迷你图可在日线与周线之间切换；后端 `/api/quotes/batch/charts` 支持 `timeframe`，并补齐周线 `EMA5/EMA10`。

## 设计结论
- 前端增加全局 `timeframe` 状态（`1D` / `1W`）。
- 图表数据缓存按 timeframe 分桶，避免切换后互相覆盖。
- 后端批量图表接口增加 `timeframe` 入参，按 `df` 或 `df_weekly` 构建 mini candles。
- 周线指标计算补充 `EMA5`/`EMA10`，保证短均线模式在周线可用。

## 兼容性
- 默认 `timeframe=1D`，不影响现有调用。
- 日线逻辑与现有接口返回保持一致。

## 验证标准
- 列表页切换到周线后，所有已显示 mini 图均可切到周线数据。
- 周线模式下 EMA 5/10、20/50 与 MACD 均可显示。
- 后端测试与前端构建通过。
