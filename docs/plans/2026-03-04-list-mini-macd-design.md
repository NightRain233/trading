# 列表迷你图显示 MACD 设计

## 目标
在前端列表点击显示 EMA 图时，同时显示 MACD 副图（柱状图 + DIF/DEA），并保持现有 EMA 交互不变。

## 设计结论
- 数据来源使用后端 `/api/quotes/batch/charts`，统一指标口径。
- 后端 `mini candles` 增加 `macd_dif/macd_dea/macd_hist` 字段。
- 前端 `MiniChart` 从单图改为上下双图：主图 K+EMA，副图 MACD。
- 缺失指标值继续按有限数过滤，不影响渲染。
- Tooltip 增加当前点位的 MACD 值。

## 约束与风险
- 列表股票较多时图表实例数增加，需要控制副图高度并避免多余重渲染。
- MACD 缺值区间允许断线/空柱，不做强行补值。

## 验证标准
- 开启列表图后，单行可同时看到 EMA 主图与 MACD 副图。
- 数据缺值时无崩溃、无控制台错误。
- 后端与前端构建/测试通过。
