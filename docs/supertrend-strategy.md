# SuperTrend 策略研究笔记

## 策略原理

SuperTrend 是基于 ATR 的动态趋势跟踪指标：

```
中轨 = (High + Low) / 2
上轨 = 中轨 + multiplier × ATR(length)
下轨 = 中轨 - multiplier × ATR(length)

方向 = +1（多头）：价格在下轨上方
方向 = -1（空头）：价格在上轨下方
```

**买入**：方向从 -1 翻 +1（趋势转多）  
**卖出**：价格跌破当前 SuperTrend 线，或方向翻 -1

止损线是**动态跟踪**的——趋势走得越远，止损线跟得越高，不会过早离场。

默认参数：`length=7, multiplier=3.0`（pandas-ta 标准默认值）

---

## 与 BB 回踩策略的对比

BB 回踩策略要求：挤压 → 突破 → 回踩 → 再涨，链条长、信号少、对标的要求高。  
SuperTrend 只看当前趋势方向，适应性更强，几乎所有有趋势的标的都能用。

| 指标 | BB 回踩 + ATR止损 | SuperTrend 日线 |
|------|-----------------|----------------|
| 交易次数（美股+加密+黄金，5年） | 80 | 149 |
| 胜率 | 36% | **60%** |
| 均收益/笔 | +4.73% | **+8.62%** |
| 平均盈利 | +19.55% | +16.46% |
| 平均亏损 | -3.70% | **-3.35%** |
| 盈亏比 | 5.29x | 4.92x |
| 组合5年收益 | +101% | **+799%** |
| 最大回撤 | 10.5% | **5.7%** |

SuperTrend 组合收益更高的核心原因：**资金利用率更高**（交易次数多，复利效应强），而不是单笔收益更大。

## 日线 / 周线 / 共振对比

标的池：NVDA / GOOGL / GC=F / BTC-USD / ETH-USD / SPY / QQQ，回测区间 2021-2026。

| 维度 | 交易次数 | 胜率 | 均收益/笔 | 组合收益 | 最大回撤 |
|------|---------|------|---------|---------|---------|
| 日线 ST(7,3) | 149 | 60% | +8.62% | **+799%** | 5.7% |
| 周线 ST(7,3) | 17 | 82% | +57.8% | +347% | 6.5% |
| 日线+周线共振 | 89 | 56% | +8.55% | +302% | **4.3%** |

**结论：日线就够了。**

- 周线胜率最高（82%），但一年只有 3-4 笔，资金大部分时间闲置，组合收益反而低
- 共振过滤掉了太多有效信号，收益从 +799% 降到 +302%
- 如果用户明确"不想频繁操作"，可以用周线版本

## 在 A 股 ETF 上的表现

标的池：17 只 A 股 ETF，回测区间 2021-2026，日线 ST(7,3)。

**汇总**

| 指标 | 值 |
|------|-----|
| 总交易次数 | 113 |
| 胜率 | 56% |
| 均收益/笔 | +5.32% |
| 平均盈利 | +11.91% |
| 平均亏损 | -2.99% |
| 盈亏比 | 3.98x |
| 组合收益（5仓） | **+172%** |
| 最大回撤 | **5.45%** |

**按年分布**

| 年份 | 笔数 | 胜率 | 均收益 |
|------|------|------|--------|
| 2021 | 14 | 57% | +3.52% |
| 2022 | 24 | 46% | +2.00% |
| 2023 | 22 | 45% | +4.13% |
| 2024 | 23 | 70% | +9.06% |
| 2025 | 25 | 68% | +8.51% |

**2022 年熊市仍有 46% 胜率、均收益 +2%**，说明 SuperTrend 在 A 股熊市里也能通过捕捉反弹趋势保持正期望。这是 BB 回踩策略做不到的（2022 年 BB 回踩几乎没有信号）。

**标的贡献（前8）**

## 为什么 SuperTrend 有效但不是"万能"

**有效的原因**：动态跟踪止损 + 趋势方向过滤，天然实现"截断亏损、让利润奔跑"。

**局限性**：

1. **参数敏感**：(7,3) 是历史最优参数，换成 (10,2) 或 (5,4) 结果差异显著，存在过拟合风险
2. **震荡市频繁止损**：横盘震荡时 SuperTrend 方向频繁翻转，会产生连续小亏
3. **执行风险**：胜率 60% 意味着连亏 3-4 笔是常态，实盘中需要严格纪律
4. **回测滑点**：代码用开盘价入场，实盘信号触发时价格可能已偏移 1-2%

## 实现说明

`backtest.py` 中的 `run_supertrend_backtest()` 函数：
- 使用 `pandas_ta.supertrend()` 计算指标
- 支持 `filter_weekly_df` 参数实现日线+周线共振过滤
- 入场：次日开盘价（含滑点）
- 止损：当日 Low 触及 SuperTrend 线时以 `min(Open, ST线)` 离场
- 手续费：双边 5bps（可配置）
