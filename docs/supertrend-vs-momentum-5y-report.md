## 假设

- 日线 SuperTrend 翻多买入，日线翻空或触及 SuperTrend 止损卖出，并用周线 SuperTrend 多头过滤。
- 当前代码中的 RS Rotation：60 日相对强弱排名，20 个交易日调仓，按资产类别月 MACD 过滤。
- ADX 研究只作为入场过滤：SuperTrend 翻多时要求 ADX 达到阈值，离场和止损不受 ADX 阻止。
- 周线 BB 对照使用 `weekly_bb_breakout_ma30`：周线突破优先，没有突破时接受突破后的回踩确认，最大持仓 90 天。
- 单边滑点 5.0 bps，买卖手续费合计按单边 5.0 bps 估算。

## 结论解读

- 基础 SuperTrend 是当前样本里最适合做默认观察页的策略：收益回撤比在中国、美股、BTC、全球观察池都优于 RS 动量轮动。
- 单纯 ADX 入场过滤没有增强 SuperTrend。它降低了回撤，但砍掉的趋势行情更多，收益回撤比反而下降。
- 周线 BB 突破+回踩更像低频主题机会策略，不适合作为普通人的默认入口；在这批样本中，中国 ETF 和全球池表现明显弱于 SuperTrend。
- 美股买入持有等权收益很高，主要来自强趋势大牛股；但这依赖择股和持有承受力，不等同于低维护策略更优。

## 可重复脚本

本报告由 `backend/strategy_comparison.py` 生成。脚本会复用本地 parquet 缓存，统一比较 SuperTrend、RS 动量轮动、买入持有、ADX 过滤 SuperTrend、周线 BB 突破+回踩。

运行：

```bash
cd backend
PYTHONPATH=. uv run python strategy_comparison.py --format markdown
```

输出 JSON：

```bash
cd backend
PYTHONPATH=. uv run python strategy_comparison.py --format json
```

换区间：

```bash
cd backend
PYTHONPATH=. uv run python strategy_comparison.py --start 2022-01-01 --end 2026-05-30
```

更完整的使用说明和后续复盘建议见 [策略回测复盘指南](strategy-backtesting-guide.md)。
