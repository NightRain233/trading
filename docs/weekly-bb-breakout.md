# 周线 BB 突破策略

## 策略口径

- **布林带**：周线 BB(20, 2)，只使用上轨和下轨参与信号判断。
- **BB 中轨（MA20）**：用于突破后的回踩确认，也作为前端图表可选展示。
- **趋势过滤**：30 周均线 MA30，收盘须在 MA30 之上才允许入场。
- **目标**：捕捉周线长期挤压后放量突破上轨的趋势段，同时允许突破后的第一次质量回踩介入。

## 入场条件

### 突破入场（同时满足）

1. 当前周线收盘 > BB 上轨
2. 当前周线收盘 > MA30
3. 近 8 周内出现过带宽收窄（带宽 < 近 8 周均值的 80%）

### 回踩入场（共同前提）

1. 最近 8 周内出现过周线收盘突破 BB 上轨，且突破周收盘 > MA30
2. 当前周线收盘 > MA30

满足共同前提后，以下任一回踩形态触发：

- **周回踩（`weeklyPullback`）**：当前周线最低价回踩到 BB 中轨 / 周 MA20 附近（默认容忍 3%），且当前周线收盘重新站回 BB 中轨 / 周 MA20。
- **日回踩（`dailyPullback`）**：最近 10 个交易日最低价回踩到日 EMA20 附近（默认容忍 1%），且最新日线收盘重新站上日 EMA20 和日 MA30。

回踩入口只作为突破后的二次介入，不单独抄底；周 MA30 仍然是趋势过滤和主要止损线。日回踩比周回踩更早，适合强趋势；周回踩更保守，价格更接近中期成本区。

## 离场条件（任一触发）

- 收盘跌破 MA30（趋势破坏）
- 收盘回到 BB 上轨以内，且上轨斜率走平或向下（动能衰竭）

## 扫描状态说明

| 状态 | 含义 |
|------|------|
| **突破** | 满足全部入场条件，策略入场信号 |
| **回踩** | 最近出现过周线突破，当前出现周回踩或日回踩确认 |
| **挤压** | 当前带宽 < 近 20 周均值的 85%，布林带收窄蓄力，等待方向选择 |
| **离场** | 触发离场条件，持仓应考虑退出 |
| **观望** | 不满足以上任何条件 |

典型节奏：**挤压 → 突破（直接入场） / 等回踩 → 回踩（确认入场） → 离场（退出）**

## 前端入口

Header 导航栏点击「周线BB」标签（靛蓝色），进入九宫格扫描页。

### 工具栏开关

| 开关 | 说明 |
|------|------|
| 全部 / 突破 / 回踩 / 挤压 / 离场 | 按信号状态筛选 |
| BB中轨 | 主图叠加 BB 中轨（紫色虚线），默认隐藏 |
| MA5 | 主图叠加周 MA5（绿色细线），默认隐藏 |
| MACD | 主图下方展示周 MACD 副图（柱状 + DIF/DEA 线），默认隐藏 |

### 图表图例

| 颜色 | 含义 |
|------|------|
| 紫色实线 | BB 上轨 / 下轨 |
| 紫色虚线 | BB 中轨（可选） |
| 琥珀色 | MA30 |
| 绿色细线 | MA5（可选） |
| 蓝色 / 橙色 | MACD DIF / DEA（副图可选） |
| 绿/红柱 | MACD Histogram（副图可选） |

九宫格按信号状态排序（突破 → 回踩 → 挤压 → 离场 → 观望），每格显示最近 52 周 K 线。

`GET /api/weekly-breakout/scan`

扫描 watchlist 所有标的，返回数组，每项包含：

```json

  "symbol": "510300.SS",
  "alias": "沪深300",
```

## 回测

策略版本 ID：`weekly_bb_breakout_ma30`，可在回测页面选择。

- 止损：跌破 MA30；回踩入口同样以周 MA30 作为主要防线
- 止盈：无固定目标，持有至离场信号触发（`exit_mode: warn_exit`）
